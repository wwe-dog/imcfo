# Experiments

Experiments are directions that may be explored, tested, revised, or discarded.

Experiments are not core rules. They must not override Constitution v2.

Use this folder for:

- AI input experiments
- visual experiments
- report interpretation experiments
- retention and review experiments
- new product surfaces
- future platform ideas
- commercial model exploration

Promotion path:

```text
experiment
-> evidence / prototype
-> critique
-> spec or ADR
-> doctrine or contract if stable
-> constitution only if it becomes a true invariant
```
