# 我为 CFO · V0.1 报表规则

本文档定义三大报表、核心指标和计算函数边界。

## 1. 总原则

- 报表计算函数必须是纯函数。
- 报表函数不直接读取或写入 AsyncStorage。
- 报表函数不依赖 React 状态。
- 简易版和专业版共享同一套数据、同一套计算逻辑。
- UI 只能展示结果，不能发明公式。

## 2. 首页六要素

首页必须展示：

1. 资产
2. 负债
3. 所有者权益
4. 收入
5. 费用
6. 利润

公式：

- 所有者权益 = 资产 - 负债
- 利润 = 收入 - 费用

## 3. 资产负债表

回答：

我现在拥有多少，欠了多少，真正属于我的钱是多少？

核心公式：

- 资产 = 负债 + 所有者权益
- 所有者权益 = 资产 - 负债

简易版展示：

- 我拥有的钱和资产
- 我欠的钱
- 真正属于我的钱
- 资产负债率

专业版展示：

- 流动资产
- 非流动资产
- 流动负债
- 非流动负债
- 所有者权益（个人净资产）

## 4. 利润表

回答：

我这个期间赚了还是亏了？

核心公式：

- 利润 = 收入 - 费用

简易版展示：

- 本期赚了多少钱
- 本期花了多少钱
- 本期盈利或亏损多少钱
- 储蓄率

专业版展示：

- 工资薪金收入
- 生活费收入
- 奖学金
- 劳务或兼职收入
- 投资收益
- 生存费用
- 居住费用
- 餐饮费用
- 交通通讯费用
- 学习成长费用
- 财务费用
- 税费
- 本期利润

## 5. 现金流量表

回答：

我的现金是怎么流进和流出的？

核心公式：

- 现金净变化 = 经营活动现金流净额 + 投资活动现金流净额 + 筹资活动现金流净额
- 期末现金余额 = 期初现金余额 + 现金净变化

现金流分类：

- `operating`
- `investing`
- `financing`
- `nonCash`

`nonCash` 不进入现金净变化。

## 6. 核心指标

V0.1 必须支持：

1. 总资产
2. 总负债
3. 所有者权益，个人净资产
4. 本期收入
5. 本期费用
6. 本期利润
7. 本期现金净变化
8. 经营活动现金流净额
9. 投资活动现金流净额
10. 筹资活动现金流净额
11. 资产负债率
12. 储蓄率

指标公式：

- 总资产 = 所有资产项目合计
- 总负债 = 所有负债项目合计
- 所有者权益 = 总资产 - 总负债
- 利润 = 收入 - 费用
- 现金净变化 = 经营活动现金流净额 + 投资活动现金流净额 + 筹资活动现金流净额
- 资产负债率 = 总负债 / 总资产
- 储蓄率 = 利润 / 收入

边界：

- 总资产为 0 时，资产负债率应显示为不可计算或 0，不应报错。
- 收入为 0 时，储蓄率应显示为不可计算或 0，不应报错。

## 8. 验收样例

最小样例：

- 现金资产 10000
- 信用卡负债 2000
- 本月工资收入 8000
- 本月餐饮费用 1200

期望：

- 总资产 = 10000
- 总负债 = 2000
- 所有者权益 = 8000
- 本期收入 = 8000
- 本期费用 = 1200
- 本期利润 = 6800

信用卡消费样例：

- 信用卡消费餐饮 100

期望：

- 费用增加 100
- 负债增加 100
- 经营活动现金流不因这笔消费减少

信用卡还款样例：

- 使用银行卡还信用卡 100

期望：

- 现金资产减少 100
- 信用卡负债减少 100
- 筹资活动现金流出 100，或展示为债务偿还现金流出
