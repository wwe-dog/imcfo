# 我为 CFO · 个人税务参考口径 v0.1

本文档整理 V0.1 的个人税务记录边界。它只用于产品字段预留和用户理解，不用于正式个人所得税申报。

## 1. 基本定位

V0.1 不做正式纳税申报、不做税务筹划、不做节税建议、不生成税务机关可提交文件。

V0.1 只做：

- 税务相关字段预留
- 税前税后收入记录预留
- 个税、社保、公积金、税费记录预留
- 自然人收入类型分类预留

## 2. 参考来源

截至 2026-04-27，V0.1 主要参考：

- 《中华人民共和国个人所得税法》
- 国家税务总局关于个人所得税的公开说明和办税口径
- 自然人收入类型的常见分类

官方来源索引：

- 国家税务总局：[https://www.chinatax.gov.cn/](https://www.chinatax.gov.cn/)
- 中国政府网：[https://www.gov.cn/](https://www.gov.cn/)

## 3. V0.1 可记录的自然人收入类型

V0.1 可记录：

- 工资薪金
- 劳务报酬
- 稿酬
- 特许权使用费
- 利息、股息、红利
- 财产租赁
- 财产转让
- 偶然所得
- 其他普通收入
- 简单兼职或副业收入

重要边界：

简单兼职或副业收入在 V0.1 只做普通自然人收入记录，不进入个体工商户经营所得核算。

## 4. 税务字段预留

V0.1 可预留：

- `taxCategory`
- `taxWithheld`
- `taxPeriod`
- `beforeTaxAmount`
- `afterTaxAmount`
- `socialInsuranceAmount`
- `housingFundAmount`
- `deductionType`
- `isTaxRelated`

这些字段可以暂时不完整使用，但不应阻碍后续 V0.2 自然人税务与年度汇总增强。

## 5. 税前税后收入规则

V0.1 可支持两种记录方式：

- 用户只知道到账金额：按税后收入记录。
- 用户知道工资条信息：可同时记录税前收入、个税、社保、公积金和税后到账金额。

产品展示应避免给出正式纳税判断。

## 6. 不做事项

V0.1 不做：

- 自动生成纳税申报表
- 税务筹划
- 节税建议
- 避税建议
- 个体工商户经营所得核算
- 增值税计算
- 发票抵扣
- 税务机关可提交文件
- 境外所得申报判断

## 7. 合规声明

产品必须明确：

我为 CFO 可以帮助用户记录和理解个人税务相关数据，但不能替代税务机关官方系统、税务师、会计师或法律专业人士。

涉及正式申报、税务争议、复杂经营所得、境外所得、个体工商户、个人独资企业等场景时，用户应以税务机关官方口径和专业意见为准。
