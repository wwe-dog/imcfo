# 我为 CFO · 准则与法规参考

本目录保存 V0.1 使用的会计、税务、合规参考口径。

这些文档是产品规则和开发规则，不是正式法律意见、税务意见、审计意见或纳税申报依据。

## 文件

- `01-accounting-standards-baseline.md`：会计准则参考口径和个人场景适配规则。
- `02-tax-baseline.md`：个人所得税相关记录边界和字段预留。
- `03-compliance-boundaries.md`：隐私、安全、税务、投资、贷款、AI 等边界。
- `imcfo-visual-system.md`：IMCFO 暗黑液态 CFO 风格、颜色治理、模块适用范围和动效边界。

## 使用原则

- 先遵守 `docs/00-project-constitution.md`。
- 再检查本目录对应专题。
- V0.1 只做普通自然人个人经营账本。
- 涉及正式税务、法律、审计、投资、贷款结论时，产品必须提示用户以官方系统和专业意见为准。
- 视觉设计先遵守 `imcfo-visual-system.md`，但不得突破项目宪法、会计规则和 V0.1 产品边界。
