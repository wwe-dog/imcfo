# 我为 CFO · 合规与安全边界 v0.1

本文档定义 V0.1 的合规、安全、风险提示边界。

## 1. 本地优先

V0.1 默认数据只存储在用户本地浏览器。

V0.1 不做：

- 登录
- 后端
- 数据库
- 云同步
- 第三方 API
- 银行卡自动同步

## 2. 最小收集

V0.1 不收集：

- 身份证号
- 银行卡号
- 家庭住址
- 人脸信息
- 通讯录
- 精确定位
- 真实征信信息

## 3. 用户控制

用户必须可以：

- 导出本地数据
- 清空本地数据
- 查看本地数据说明

清空数据必须二次确认。

导出数据必须包含：

- 账户
- 交易
- 资产
- 负债
- 报表配置
- version

## 4. 税务边界

V0.1 不提供：

- 正式税务申报
- 税务筹划
- 节税建议
- 避税建议
- 税务结论
- 税务机关可提交文件

涉及正式申报或税务争议时，必须提示用户以税务机关官方口径和专业意见为准。

## 5. 投资与贷款边界

V0.1 不提供：

- 投资建议
- 股票基金推荐
- 贷款推荐
- 信用评估
- 法律意见

报表可以展示事实和用户录入数据的计算结果，但不应给出买卖、借贷或投资决策建议。

## 6. AI 边界

AI 不是 V0.1 功能。

后续如接入 AI，必须满足：

1. 用户明确授权后才可分析数据。
2. AI 结论只作为参考。
3. AI 不构成投资、税务、法律或贷款建议。
4. AI 不得编造用户没有记录的数据。
5. AI 分析必须可追溯到具体数据。

## 7. 修改与删除

V0.1 可以允许自由修改和删除记录。

底层应预留：

- `createdAt`
- `updatedAt`
- `source`
- `version`
- `auditTrail`
- 软删除
- 冲销记录
- 更正分录
- 操作日志
