Important product boundaries:
- This is 我为 CFO, a personal finance app that treats an ordinary natural person as a personal company.
- Do not turn it into a normal bookkeeping app.
- Do not add backend, login, database, cloud sync, real API integrations, AI analysis, payment, membership, VAT, invoice logic, individual-business-owner logic, or formal tax filing.
- Keep user-facing text Chinese.
- Keep bottom navigation as 首页 / 管理 / 报表 / 我的.

Important architecture boundaries:
- UI screens must not call AsyncStorage directly.
- Storage adapters must stay separate from accounting/report calculations.
- Report calculation functions must remain pure and testable.
- UI must not invent accounting formulas.
- Do not change accounting formulas, storage schema, transaction mapping, natural language parser behavior, route keys, storage keys, or domain model fields unless explicitly required.
