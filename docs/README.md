# IMCFO Docs

This docs tree uses the IMCFO Constitution v2 Council workflow.

## Current Structure

- `council/`: council agent definitions, protocols, execution records, and QA reports.
- `constitution/`: living constitution, doctrines, contracts, and change governance.
- `discovery/`: current codebase facts. Discovery is not permanent law.
- `specs/`: current product, UI, interaction, and implementation specs.
- `experiments/`: exploratory directions that may be tried or discarded.
- `adr/`: major decision records.
- `context/`: current project continuity notes for long tasks and context compression.

## Deprecated

Legacy `docs/00-10` are intentionally deprecated and must not be restored unless explicitly requested by the user.

Current implementation facts must be rediscovered from code and recorded under `docs/discovery/`.
