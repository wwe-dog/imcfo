## Council Principle

宪法不是为了禁止变化，而是为了保证变化不会破坏 IMCFO 的灵魂。

Constitution v2 将采用 Codebase-first + Living Constitution 思路：

- 当前代码、配置、目录结构只作为 current implementation facts。
- Constitution v2 保护产品灵魂、财务底线、AI 入账边界、架构契约、协作机制和变更机制。
- Constitution v2 不锁死技术栈、不锁死页面结构、不锁死视觉实现、不锁死 API / 模型供应商。
