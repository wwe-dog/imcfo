# IMCFO Usage Rules

使用 Arco Design Pro 2.0 作为参考时，必须遵守以下规则：

1. 不得直接复制 Arco 的桌面端布局。
2. 不得把 Arco UI、Arco Pro、Web/Admin 依赖引入 React Native App。
3. 不得新增 Web 管理后台依赖。
4. 该资源仅作为视觉和结构参考。
5. 所有实现必须保持 Expo React Native mobile-first。
6. 应用视觉应继续保持当前 cream / green / purple 方向。
7. 应用 Arco 参考时不得改变会计公式、交易规则或存储 schema。
8. 每次只优化一个页面或一个局部模块，避免整站式重构。
9. 首页应参考 dashboard 的信息层级，不复制桌面多列布局。
10. 交易记录应参考表格密度，但转换为移动端账本行。
11. 账户、资产、负债管理应参考管理页结构，但采用移动端 drilldown。
12. 报表页应参考图表和数据卡片组合，但保持三大报表逻辑不变。
