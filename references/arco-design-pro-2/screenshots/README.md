# Screenshots

当前 Codex 会话没有可用的 Figma export 工具，无法自动导出 Arco Design Pro 2.0 截图。

请手动从 Figma 导出关键截图并放入本目录。

建议文件名：

- `dashboard-reference.png`
- `data-visualization-reference.png`
- `table-reference.png`
- `detail-page-reference.png`
- `form-reference.png`

导出建议：

- 优先选择能体现数据层级和管理页结构的 frame。
- 不需要保存所有页面，只保存对“我为 CFO”移动端设计有启发的关键截图。
- 截图仅作为视觉参考，不代表可以直接复制桌面端布局。
