# Arco Design Pro 2.0

## 资源链接

Figma Community: https://www.figma.com/community/file/1065157704393387989/arco-design-pro-2-0

## 在我为 CFO 中的用途

本目录用于保存 Arco Design Pro 2.0 的本地设计参考。它可以作为“我为 CFO”后续移动端界面优化时的视觉参考，重点参考：

- 数据仪表盘的信息层级
- 密集表格和列表的排版节奏
- 图表卡片的组合方式
- 管理页的信息组织
- 详情页的字段层级

## 重要警告

Arco Design Pro 是桌面端 / 管理后台方向的设计资源；“我为 CFO”是 Expo React Native 移动端个人财务 App。后续实现时不能直接复制桌面端布局，也不能把 Arco Web/Admin UI 库引入移动端项目。

## 使用方式

Codex 或设计人员可以从该资源中抽取布局节奏、卡片层级、表格密度、图表样式和管理页模式，再移动端化改造到“我为 CFO”。

优先用于这些页面：

- 首页数据仪表盘
- 交易记录账本列表
- 账户管理
- 资产负债管理
- 报表页
- 我的 / 数据管理
